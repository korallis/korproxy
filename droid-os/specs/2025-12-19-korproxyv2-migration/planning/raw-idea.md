# KorProxyV2 Migration

## Raw Feature Description

Migrate KorProxy to the KorProxyV2 stack:
- Simplify to macOS app initially, with plan to expand to Windows
- Preserve existing subscriptions/accounts and licensing checks
- Update CI/CD and code signing configuration
- Maintain compatibility with existing user data and authentication

## Context

This migration involves transitioning from the current KorProxy architecture to a simpler, more maintainable stack (KorProxyV2) starting with macOS support, while ensuring no disruption to existing users' subscriptions, licenses, and accounts.
